Description

维护一棵支持插入节点与求$k$个节点的$LCA$的树。

总节点数 $n\leq 3 \times 10^6$ ，操作次数 $m \leq 10^3$，$k \leq 10^3$

题目链接： BZOJ P5165

Solution

思路题目名称已经给了好嘛。。。

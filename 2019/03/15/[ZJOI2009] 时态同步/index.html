「ZJOI2009」时态同步

P_Wang

2019-03-15

OI / 题解

Description

一棵有$N$个结点，带边权的有根树，求至少需要增加多少边权才能使根结点到叶子结点的所有路径的权值和相同。

对于$100\%$的数据，$N \leq 500000$，边权$t_e \leq 1000000$。

题目连接戳这里

Solution

树形$DP$的板子题。

令$dp_x$为叶子结点到$x$结点的最大时间。

则有 $dp_x = \max (dp_{son[x]} + dis_{x,\ son[x]})$

因此 $ans = \sum (dp_x - dp_{son[x]} - dis_{x,\ son[x]})$

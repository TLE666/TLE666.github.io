[SCOI2009] windy 数

 2019-03-10
 

Description

求$A$，$B$之间所有满足相邻两位之间差$\geq2$的整数。

对于$100$%的数据，

$1 \leq A \leq B \leq 2 \times 10^9$。

Solution

数位$DP$模板题。

令$dp_{i,j}$表示第$i$位，上一位为$j$的总方案数（一直到结束）。

然后逐位$DP$即可。

 2019-03-17
 

Description

给一段可能有负数的序列，查询最大子段和。

对于$100 \%$的数据，序列长度$N \leq 50000$，$a[i] \leq 15007$

题目链接戳这里

Solution

线段树查询最大子段和。

这里需要维护四个信息：

  • $s$ 区间和
  • $mx$ 区间最大子段和
  • $lmx$ 区间前缀和最大值
  • $rmx$ 区间后缀和最大值

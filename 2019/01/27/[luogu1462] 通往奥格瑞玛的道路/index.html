「luogu1462」通往奥格瑞玛的道路

 2019-01-27
 

Description

$n$个点$m​$条边的带权无向图

每个点还有一个点权

求$1-n$的一条最短路径，使得路径长度比$b​$短，并且经过的点权之和最大。

输出这个最大值

看完题面感觉语文要挂科。。。o(╥﹏╥)o

Solution

这题正解 $SPFA\ +​$ 二分

二分答案，$check()$函数中过一遍$SPFA$，然后跑一下$path$，也就是回溯函数（用来求最短路径），然后判一下符不符合题意就行了
